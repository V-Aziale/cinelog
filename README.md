# CineLog

A personal media watchlist tracker for logging and managing movies, anime, TV shows, Asian dramas, and documentaries. It aims to develop a data entry and file handling module that allows users to add, edit, and delete media records saved to a local file, implement a search and filter function that uses loops and conditionals to return records matching user-specified parameters, and build a statistics module that uses iteration to calculate and display summary data such as total entry count, status breakdown, and average rating.

## Features
- Add, edit, and delete media entries
- Search by title keyword
- Filter by watch status
- Statistics overview

## How to Run
g++ cinelog.cpp -o cinelog
./cinelog

## Status
Work in progress — not all features are implemented yet.
